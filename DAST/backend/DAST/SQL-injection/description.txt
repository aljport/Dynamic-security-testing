This Python script is an asynchronous web scanner designed to identify various SQL injection vulnerabilities, including error-based, boolean-based, union-based, and time-based attacks. It utilizes aiohttp and asyncio to concurrently crawl a target domain, extract internal links and HTML forms, and systematically test parameters with a library of predefined payloads. The tool determines potential vulnerabilities by analyzing server response times, searching for specific database error strings, and performing similarity comparisons between baseline and injected responses.
